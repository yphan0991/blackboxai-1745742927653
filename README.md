# FurniLux

## Project Overview
FurniLux is a modern furniture sales website dedicated to providing elegant and sustainable furniture designed to elevate your living space. The project focuses on using eco-friendly materials and innovative craftsmanship, inspired by the sleek aesthetics of modern design brands. This website serves as an online platform to showcase featured products, along with an introduction to the company's mission and vision.

## Installation
To run this project locally, follow these steps:

1. **Clone the repository**:
   ```sh
   git clone <repository-url>
   ```
2. **Navigate to the project directory**:
   ```sh
   cd furnilux
   ```
3. **Open the index.html file** in your preferred web browser:
   ```sh
   open index.html   # For Mac users
   start index.html  # For Windows users
   ```

## Usage
Simply open the `index.html` file in a web browser to view the FurniLux website. Explore the featured products, learn about the project, and find contact information in the footer section.

## Features
- **Modern Design**: A contemporary layout that provides a visually appealing user experience.
- **Responsive Navigation**: A mobile-friendly navigation menu that adapts to different screen sizes.
- **Product Showcase**: Section displaying featured products with images, descriptions, and prices.
- **Project Introduction**: Detailed section explaining the company’s mission and sustainable practices.
- **Contact Information**: Easy access to company's contact details and social media links.

## Dependencies
This project uses the following external libraries and frameworks:
- **Google Fonts**: For custom fonts (`Roboto`).
